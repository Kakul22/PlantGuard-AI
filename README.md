# 🌿 PlantGuard AI

An AI-powered web application that detects plant diseases from leaf images and helps farmers and users take early preventive actions.

---

## 🚀 Features

- 🌱 Detect plant diseases using image input
- 🤖 Deep Learning model for accurate predictions
- 📊 Fast and real-time results
- 🌍 User-friendly web interface
- 🔒 Secure and efficient processing

---

## 🧠 Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Flask (Python)
- **Machine Learning:** TensorFlow / Keras
- **Libraries:** NumPy, OpenCV
- **Dataset:** Plant Village Dataset

## ⚙️ Installation
# Clone the repository
git clone https://github.com/Kakul22/PlantGuard-AI.git

# Navigate to project folder
cd PlantGuard-AI

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py

# Open in browser
http://127.0.0.1:5000/

## How it Works 
1. Upload image
2. Model processes it
3. Prediction shown
